StreamVault Local GitHub Repository

By Aditya Priyadarshi

Minimal local video downloading.

StreamVault Local is a clean Python and Flask web app that runs on your machine, accepts a video URL, and saves the best available MP4 video with audio.

Read setup guide View on GitHub

Local first

Your downloads are saved directly on your computer in the project downloads folder.

High quality

Uses yt-dlp to select the best MP4 video and audio streams, then merges them automatically.

GitHub ready

Includes a professional README, license, requirements file, and GitHub Pages landing page.
